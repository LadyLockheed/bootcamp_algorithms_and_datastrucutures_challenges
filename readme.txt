1. skapa en ny mapp var som helst på datorn.
2. Döp den till nåt käckt, tex Sandbox.
3. Öppna den mappen i vs code.
4. Skapa en fil som heter tex main.js
5. Lägg en console.log i den filen
6. I terminalen, tex Windows powershell (se till att gå in i den nya mappen)
skriv "node main.js" eller vad man nu döpte filen till.
7. Se resultatet av console.log i terminalfönstret.

Obs. För att kunna köra node måste node.js vara installerat.
